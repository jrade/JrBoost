Before running the Higgs examples you must

1) download the data files training.csv and test.csv from www.kaggle.com/c/higgs-boson/data
2) set the variable dataDirPath in higgs_util.py to the path of the directory that contains these files
3) set the variable resultDirPath in higgs_util.py to the path to the directory where the result file should be saved

The saved result file can be uploaded to www.kaggle.com/c/higgs-boson/submit and scored.
